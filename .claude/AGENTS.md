# Keycloakify Agents & Skills Guide

Specialized agents and skills to accelerate common development tasks. Type `/` in Claude Code to load one.

## ⚡ Quick Reference

| Command | Type | Use When |
|---------|------|----------|
| `/grill-me` | Skill | Interview the user relentlessly about a plan or design until reaching shared understanding, resolving each branch of the decision tree. |

---

## Tips

- Load the skill early — the more context you provide, the better the output
- Reference existing components/stores as examples when prompting
- Skills support clarifying questions — iterate until the output is right
